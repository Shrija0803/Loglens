
Project Name: LogLens – Simplified Security Log Analysis Tool

Presented by-Shrija Chalke, shrijachalke@gmail.com, (Cybersecurity)


Description:
LogLens is a web-based tool that analyzes web server log files to detect suspicious activities such as SQL injection, XSS, and unauthorized access attempts.

Steps to Run:

1. Install Python
2. Install required packages:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser and go to:
   http://localhost:5000
5. Upload the demo.log file to view analysis
